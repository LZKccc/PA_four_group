## 📑 项目简介 (Project Overview)
本项目旨在通过对国家统计局提供的宏观经济数据、地方财政收支数据以及房地产市场数据进行深度清洗与实证分析，探究中国主要城市在 2006-2024 年间的地方财政健康状况（预算缺口率 Gap/GDP）。项目重点分析了区域性差异（长三角 vs 珠三角）、一线城市财政周期的同频共振，并以数据实证了“土地财政”对地方预算缺口的显著影响。

## 📂 目录结构 (Directory Structure)
本项目严格遵循数据科学与学术可复现的标准工程规范，构建了自动化的数据处理工作流：

* 📁 **`data_raw/`**：存放原始下载的未清洗数据（支持国家统计局标准的宽表格式导入）。
* 📁 **`data_clean/`**：存放经过宽表转长表（Melt）、多键合并（Merge）和空值处理后生成的标准化面板数据（`city_master_data_final.csv`）。
* 📁 **`output/`**：存放生成的统计摘要表格与达到学术期刊发表标准的高清可视化图表。
* 📝 **`01_data_clean.ipynb`**：数据清洗核心引擎（完成自动化读取、格式转换、计算环比增长率与多表串联）。
* 📝 **`02_data_analysis.ipynb`**：数据分析与可视化脚本（完成统计极值提取、区域打标分组以及高级统计回归绘图）。

## 💡 核心分析发现 (Key Findings)
本项目顺利完成了指导老师布置的所有基础与进阶分析任务：

1. **重点年份极值分析**：通过自动化遍历，提取了 2006、2010、2014、2018、2022 年的财政缺口极值。数据客观反映了中西部城市（如拉萨、西宁）较高的转移支付依赖度，以及东部数字经济强市（如杭州）优异的财政自给能力。
2. **北上广深一线城市对比**：绘制了四大一线城市的 Gap/GDP 趋势图，不仅证明了其极强的自我造血能力（常年处于财政盈余或低缺口），更直观展现了其受宏观经济周期影响的同频共振特征。
3. **区域经济底色（长三角 vs 珠三角）**：通过代码自动对目标城市进行区域（Region）打标归类并计算均值，展现了“多极均衡”的长三角与“双核驱动”的珠三角在面临不同经济周期时呈现出的差异化财政韧性。
4. **“土地财政”实证检验 (进阶)**：引入商品房销售与房地产开发数据，利用 OLS 线性回归散点图明确验证了“商品房销售面积”与“预算缺口率”之间存在显著的负相关关系。证实了房地产周期对地方一般公共预算（契税、相关增值税等）的直接支撑逻辑。

## 🛠️ 技术栈与复现指南 (Reproducibility)
* **核心工具库**：`pandas` (面板数据处理与聚合), `matplotlib` (基础可视化), `seaborn` (高级统计与回归可视化)。
* **一键复现**：环境配置完成后，无需修改路径，按顺序依次运行 `01_data_clean.ipynb` 和 `02_data_analysis.ipynb`，即可自动化复现所有中间数据集与最终图表。
